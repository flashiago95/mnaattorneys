# MNA Attorneys — Website

**Mdyesha Ndema Attorneys** | Corporate & Commercial Law  
Built on the Bustar HTML template framework with custom MNA branding.

## Project Structure

```
mna-attorneys/
├── index.html              # Homepage
├── assets/
│   ├── css/
│   │   ├── vendor/         # Bootstrap, animate.min.css
│   │   ├── plugins/        # Swiper, nice-select, etc.
│   │   └── main.css        # Custom MNA styles
│   ├── js/
│   │   ├── vendor/         # jQuery, Bootstrap bundle
│   │   ├── plugins/        # GSAP, Swiper, Lenis, WOW, etc.
│   │   └── main.js         # Custom MNA scripts
│   ├── images/             # Replace placeholders with real photos
│   └── fonts/              # Remixicon font files
```

## Setup

1. Clone the repo
2. Replace placeholder images in `assets/images/` with real photography
3. Open `index.html` in a browser — no build step required
4. Deploy to any static host (Netlify, Vercel, GitHub Pages)
